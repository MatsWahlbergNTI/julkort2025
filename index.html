Lektionerna fram till lovet går ut på att göra ett julkort i Canvas. Canvas är ett sätt att rita med vektorgrafik direkt på hemsidor.

Utgå från koden i den här html-filen och gör ett eget kort med klassiska julmönster.

  1. Välj vad du vill rita först (förslagsvis ett paket)
    Förslagsvis kan rektanglar användas för att göra paket och trianglar användas för att göra julgranar.
    För att göra ett paket kan du rita en rektangel för hela paketet och sedan två mindre rektanglar för banden.
  2. Kommentera bort rader i skriptet som inte ritar rektanglar (eller trianglar om du gör en julgran) genom att skriva // i början av raden
  3. Ändra på koden tills din hemsida ser ut som ett julkort

Hemsidan ska sedan läggas ut på GitHub och vi kollar på julkorten tillsammans den 16 december.
Skriv i uppgiften om du vill att kortet ska visas eller inte.
